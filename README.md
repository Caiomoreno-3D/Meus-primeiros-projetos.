# Sistema de Vendas - Prata & CriaRi

Meu primeiro sistema funcional de gerenciamento de pedidos, feito 100% por mim com ajuda de IA para a parte visual.

## Tecnologias usadas
- Python (lógica completa)
- Tkinter (interface gráfica)
- JSON (armazenamento simples de dados)

## Funcionalidades
- Cadastro de novo pedido
- Consulta de todos os pedidos (tabela visual)
- Alteração de pedido existente
- Geração automática de código único (ex: pedido_0001_VENDEDORA_01012026)
- Cálculo de total e status de pagamento

## Como rodar
1. Instale Python 3.10+ (se não tiver)
2. Abra o arquivo `Vendas - Pratas e CriaRi.py`
3. Rode com duplo clique ou no terminal: `python "Vendas - Pratas e CriaRi.py"`
